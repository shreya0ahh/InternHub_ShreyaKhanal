## 📋 Project Overview
InternHub is a professional networking platform connecting students with companies for internship opportunities.

## 👨‍💻 Developed by Sameer Ansari (@sameertech28)

## ✨ Features Implemented

### ✅ Authentication System
- Login/Signup with form validation
- Password strength indicator
- User type selection (Student/Company)
- Real-time validation feedback

### ✅ Navigation & Layout
- Responsive navbar with search
- Sidebar with active states
- Dropdown menus for profile & notifications
- Notification badges

### ✅ Feed System
- Create posts with image upload
- Like, comment, and share functionality
- Real-time post updates
- Delete own posts

### ✅ Profile Management
- Avatar and cover photo upload
- Profile completion progress bar
- Skills, experience, education sections
- Edit profile functionality

### ✅ Connections Network
- People You May Know suggestions
- Send/accept/reject connection requests
- View connections list
- Message connected users

### ✅ Jobs Portal
- Job listings with filters
- Post jobs (for companies)
- Apply with CV upload (for students)
- View applicants (for companies)
- Save jobs functionality

### ✅ Messaging System
- Real-time chat interface
- Conversations list
- Online status indicators
- Message history

### ✅ Search Functionality
- Global search with filters
- Search by name, skills, company
- Location and user type filters
- Highlight matching text

### ✅ Notifications
- Real-time notifications
- Unread badges
- Mark as read functionality
- Different notification types

## 🛠️ Technologies Used
- HTML5
- CSS3 (Custom Properties, Flexbox, Grid)
- Vanilla JavaScript
- LocalStorage for data persistence
- Glass morphism design
- Responsive design principles
